# SlideshowInAngular

\- Angular 18 application + Bootstrap 5.

\- Slideshow with some images.

## Installation + run app

**Command to install**

_npm install_

or shorter:

_npm i_

**Command to run the application:**

_ng serve --open_

or shorter:

_ng s --o_

## Application

A demo slideshow with some images.

Experiment with NgOptimizedImage.
